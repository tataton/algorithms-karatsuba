The Karatsuba Multiplication Algorithm

The Karatsuba Multiplication Algorithm is a recursive method for multiplying two Integers. The algorithm reduces the total number of computational steps, especially for multiplying large numbers, relative to the standard single-digit multiplication method that we all learn in grade school.

This page compares my Karatsuba algorithm output with output from big.js, a JavaScript library designed for precision arithmetic on large numbers. I can't use direct multiplication in JavaScript to test, because native JavaScript has an upper limit to integers of 253, or 9007199254740991. No JavaScript math can use a number higher than this as an input, or result in a number higher than this as a product, without introducing error. In my code, I deal with this by handling all values other than single digits as arrays of digits. I am actually not sure how big.js deals with it, but I only use big.js to test; it is not used in my Karatsuba method code. I think this type of problem would normally be tested in a language like Python, which has no limit on integer size. But I don't know how to handle web output for any other language yet, so I'm using JavaScript.
